# 0x01. NoSQL

This project covers various aspects of working with NoSQL databases, specifically MongoDB. It includes tasks ranging from basic database operations to more complex queries and data analysis.

## Files

- 0-list_databases: List all databases
- 1-use_or_create_database: Create or use a database
- 2-insert: Insert a document
- 3-all: List all documents
- 4-match: List all documents with specific criteria
- 5-count: Count documents
- 6-update: Update a document
- 7-delete: Delete documents
- 8-all.py: Python function to list all documents
- 9-insert_school.py: Python function to insert a document
- 10-update_topics.py: Python function to update document topics
- 11-schools_by_topic.py: Python function to find schools by topic
- 12-log_stats.py: Python script for log statistics
- 100-find: Advanced MongoDB query
- 101-students.py: Python function for top students
- 102-log_stats.py: Advanced log statistics

## Usage

Each script can be run individually. Python scripts require PyMongo to be installed.
